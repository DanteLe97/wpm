# Mac Live Style

Plugin WordPress cho phép tùy chỉnh giao diện trực tiếp với giao diện người dùng thân thiện.

## Mô tả

Mac Live Style là một plugin WordPress mạnh mẽ cho phép người dùng tùy chỉnh giao diện website của họ một cách trực quan và dễ dàng. Plugin này cung cấp các công cụ để thay đổi màu sắc, font chữ, và các yếu tố giao diện khác mà không cần phải chỉnh sửa code.

## Tính năng chính

- Tùy chỉnh màu sắc trực tiếp
- Thay đổi font chữ và kích thước
- Giao diện người dùng thân thiện
- Tích hợp với theme WordPress
- Hỗ trợ responsive design

## Cài đặt

1. Tải plugin lên thư mục `/wp-content/plugins/` của website WordPress
2. Kích hoạt plugin thông qua menu 'Plugins' trong WordPress
3. Truy cập vào menu 'Mac Live Style' trong admin dashboard để bắt đầu tùy chỉnh

## Yêu cầu hệ thống

- WordPress 5.0 trở lên
- PHP 7.2 trở lên
- MySQL 5.6 trở lên
